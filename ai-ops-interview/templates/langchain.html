🦜 LangChain

LLM 应用开发框架

📖 什么是 LangChain？

LangChain 是一个用于开发由语言模型驱动的应用程序的框架。它提供了模块化的抽象和工具，使开发者能够轻松构建复杂的 LLM 应用。

🎯 核心特性

🔗 链式调用

将多个 LLM 调用组合成链式工作流，实现复杂的业务逻辑。

🧠 记忆管理

内置多种记忆机制，支持对话历史管理和上下文追踪。

🔌 工具集成

丰富的工具库，可轻松集成外部 API、数据库和文件系统。

📚 文档加载

支持多种文档格式，自动分割和向量化处理。

🤖 Agent 支持

构建智能 Agent，实现自主决策和工具使用能力。

📊 评估框架

内置评估工具，量化 LLM 应用的性能和质量。

💻 快速开始

# 安装 LangChain
pip install langchain

# 基础示例
from langchain.llms import OpenAI
from langchain.chains import LLMChain
from langchain.prompts import PromptTemplate

# 初始化 LLM
llm = OpenAI(temperature=0.7)

# 创建提示模板
prompt = PromptTemplate(
    input_variables=["topic"],
    template="请详细解释一下{topic}的核心概念。"
)

# 创建链
chain = LLMChain(llm=llm, prompt=prompt)

# 执行
result = chain.run("机器学习")
print(result)

❓ 面试问题

1. LangChain 的主要组件有哪些？

参考答案：LangChain 的核心组件包括：Models（模型）、Prompts（提示）、Chains（链）、Agents（Agent）、Memory（记忆）、Tools（工具）等。

2. 如何使用 LangChain 实现 RAG（检索增强生成）？

参考答案：使用 LangChain 实现 RAG 需要：1) 加载文档（Document Loaders）2) 分割文本（Text Splitters）3) 创建向量存储（Vector Stores）4) 构建检索链（Retrieval Chains）5) 结合 LLM 生成答案。

3. LangChain 中的 Agent 是如何工作的？

参考答案：Agent 使用 LLM 来决定采取什么行动以及行动的顺序。它可以根据用户输入自主选择使用哪些工具，并解释结果。
